Welcome to AESP's documentation!
#################################

**AESP**  (Adaptive Evolutionary Structure Predictor) is a robust, open-source Python library 
for Crystal Structure Prediction. AESP is dedicated to quick and easy operations, so that people 
can get the results they want without taking too much time.

Check out the :doc:`./usage/index` section for further information, including how to :doc:`\ install` 
the project. Of course, you can also click `here <https://aesp.readthedocs.io/_/downloads/en/latest/pdf/>`_ 
for a pdf instruction.

.. note::

   The AESP is under active development.
