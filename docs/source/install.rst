
Installation
############

Requirements
============

- python 
- pymatgen
- ase
- dscribe

Installation
============

1.To use AESP, first install pymatgen and numpy using pip:

.. code:: console

   pip install numpy
   pip install pymatgen

2.Install AESP

.. code:: python

   pip install aesp

In addition, you can also visit `pypi`_ to download.

.. _pypi: https://pypi.org/project/aesp/
