Welcome to AESP’s documentation!#

AESP (Adaptive Evolutionary Structure Predictor) is a robust, open-source Python library for Crystal Structure Prediction. AESP is dedicated to quick and easy operations, so that people can get the results they want without taking too much time.

Check out the ./usage/index section for further information, including how to Installation the project. Of course, you can also click here for a pdf instruction.

Note

The AESP is under active development.
